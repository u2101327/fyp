Investigation Guide:

  1. Click "Visit Channel": Opens the Telegram channel in a new tab for manual investigation
  2. Check Leak Status: URLs marked with red badges have credential leaks detected
  3. Review Source: Click "View Source" to see where the URL was discovered
  4. Filter Results: Use the search and filter options to focus on specific channels
  5. Track Progress: All crawled URLs are saved to both Django database and OpenSearch for comprehensive tracking

Data Storage:

Crawled URLs are automatically saved to both Django database (for web interface) and OpenSearch (for advanced search and analytics). This dual storage ensures data persistence and enables powerful investigation capabilities.
